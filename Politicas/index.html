Administrar las políticas de grupo

  • El pirmer paso que debemos llevar a cabo es crear unidades organizativasbajo el dominio "QUINTERO1W.IDP" con el nombre de "OU_jedi" y "OU_sith", donde moveremos a los usuarios obiwan,quigon,yoday maul,emperador,vader respectivamente.

  • Vinculamos un GPO para cada unidad organizativa.

  • Nos situamos sobre la unidad organizativa Domain Controllers y pulsamos sobre la misma con el botón derecho del ratón para elegir la opción "Bloquear herencia".

  • En el Editor de objetos de Directiva de grupo, buscamos la Directiva de grupo que queremos modificar y en el cuadro de diálogo cambiamos la configuración. Escogimos "Habilitar Windows Messenger" y "Prohibir la instalación o desinstalación de perfiles de color"como ejemplo.

  • Quitar el menú Ejecutar del menú Inicio.

    • Vamos a: "Configuración de usuario / Plantillas administrativas / Menú Inicio y barra de tareas" y habilitamos la opción Quitar el menú Ejecutar del menú Inicio.

  • Prohibir el acceso al Panel de control.

    • Vamos a: "Configuración de usuario / Plantillas administrativas / Panel de control" y habilitamos la opción Prohibir el acceso al Panel de control.

  • Ocultar el icono Mis sitios de red del escritorio.

    • Vamos a: "Configuración de usuario / Plantillas administrativas / Escritorio " y habilitamos la opción Ocultar el icono Ubicaciones de red del escritorio.

  • Quitar el icono Mis sitios de red del menú inicio.

    • Vamos a: "Configuración de usuario / Plantillas administrativas / Menú Inicio y barra de tareas " y habilitamos la opción Quitar el icono Red del menú Inicio .

  • Quitar Conexiones de red del menú Inicio.

    • Vamos a: "Configuración de usuario / Plantillas administrativas / Menú Inicio y barra de tareas" y habilitamos la opción Quitar Conexiones de red del menú Inicio .

  • Ocultar unidades específicas en Mi PC.

    • Vamos a: "Configuración de usuario / Plantillas administrativas / Componentes de Windows / Explorador de Windows" y habilitamos la opción Ocultar estas unidades específicas en Mi PC .

  • Habilitar Quitar “Conectar a unidad de red” y “Desconectar de unidad de red”.

    • Vamos a: "Configuración de usuario / Plantillas administrativas / Componentes de Windows / Explorador de Windows" y habilitamos la opción Quitar “Conectar a unidad de red” y “Desconectar de unidad de red”.
